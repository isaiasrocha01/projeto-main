This repository contains a collection of independent mini web applications developed with **HTML**, **CSS**, and **JavaScript**.  
Each subfolder represents a different tool or feature, such as calculators, generators, calendars, and personal portfolio sections.

---

## 🧩 Project Structure

### 📁 /calculadora
A simple and functional **calculator web app** built with JavaScript.
- `calculadora.html` – main calculator page  
- `js/scripts.js` – logic for operations  
- `imagens/` – image assets used in the UI  

### 📁 /calendário_compacto
A **compact calendar tool** designed to manage events and goals.
- `calendario.html`, `day.html`, `metas.html` – calendar interface and goals page  
- `script.js` – main JavaScript functionality  
- `css/` – custom styles for the compact calendar  

### 📁 /curriculo
A **personal resume (CV) page** built as a web portfolio.
- `curriculo.html` – main resume page  
- `sobre.html` – about section  
- `script.js` – script for interactive elements  
- `styles.css` – custom layout and theme styling  

### 📁 /gerador
Contains multiple **document generators** for civil and religious records.
- `/casamento civil/` – generates civil marriage certificate  
- `/casamento religioso com efeito civil/` – generates religious marriage certificate  
- `/nascimento/` – generates birth certificate  
- `/obito/` – generates death certificate  
- `/contato/` – contact page for the generator section  
- `gerador.html` – main entry page  

Each generator folder includes:
- An HTML file (template for the document)  
- A JS file (logic and data handling)  

### 📁 /projetos
A general showcase of your web projects.
- `projetos.html` – index of your works  

### 📁 /img and /imagens
Shared assets and icons used across all sections, including:
- Calculator, calendar, GitHub, LinkedIn, and personal icons.

---

## 🛠️ Technologies Used
- **HTML5** – Structure and layout  
- **CSS3** – Styling and responsiveness  
- **JavaScript (ES6)** – Logic, interactivity, and dynamic generation

## 🚀 How to Run
1. Clone or download this repository:  
git clone https://github.com/your-username/PROJETO-MAIN
2. Open the folder in your preferred IDE (VS Code recommended).  
3. Open `index.html` in your browser to navigate between tools.  
4. Each section can also be accessed directly from its subfolder.

## 📄 Notes
- All tools are designed to work locally — no backend or server configuration required.  
- Make sure all `.js` and `.css` files maintain their relative paths when moved or deployed.
